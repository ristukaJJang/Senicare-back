회원가입(sign up)

입력 (요청)
- 이름(name) 문자열 필수
- 아이디(userId) 문자열 필수
- 비밀번호(password) 문자열 필수 8-13자리 영문+숫자 혼합
- 전화번호(telNumber) 문자열 필수 숫자만 11자
- 인증번호(authNumber) 문자열 필수 숫자만 4자

절차 (로직)
1. 유효성 검사 (유효성 검사 실패)

2. 아이디 중복 확인
2-1 아이디로 데이터베이스 조회
만약 존재하는 아이디 -> 중복된 아이디

3. 전화번호 중복 확인
3-1 전화번호로 데이터베이스 조회
만약 존재하는 전화번호 -> 중복된 전화번호

4. 전화번호 인증 확인
4-1 전화번호와 인증번호로 데이터베이스 조회
만약 존재하지 않는 전화번호와 인증번호 -> 전화번호 인증 실패
(데이터베이스 에러)

5. 비밀번호 암호화


6. 레코드 삽입
(데이터베이스 에러)

7. 성공 처리


출력 (응답)
